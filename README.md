# Expense Tracker
A personal expense tracking web app built with HTML, CSS, and JavaScript.

## Features
- Add, edit, delete, search, and filter transactions
- Data persists via localStorage (JSON)
- One-click CSV export
- Spending charts using Chart.js
- Form validation with inline errors
- Responsive layout

## How to Run
1. Clone the repo
2. Open `index.html` in any browser
